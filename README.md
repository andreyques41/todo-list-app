# Todo List App

Frontend application demonstrating modular JavaScript architecture and modern build tools.

**Author**: [Andrey Quesada](https://github.com/andreyques41) | Software Engineer | Backend Systems & AI-Assisted Development

---

## Stack

**Frontend**: HTML, CSS, Vanilla JS, Vite

---

## Features

- Task CRUD operations
- Local storage persistence
- Modular component architecture
- Responsive design
- MVC pattern implementation

---

## Architecture

```
Main/
├── main.html
├── styles/           # CSS modules
└── scripts/
    ├── core/         # App foundation
    ├── storage/      # Data persistence
    ├── models/       # Data structures
    ├── business-logic/  # Application logic
    └── ui/           # Components and views
```

---

## Quick Start

```bash
npm install
npm run dev
```

---

## What This Demonstrates

- Modular JavaScript architecture
- MVC pattern in frontend
- Component-based design
- Clean separation of concerns
- Modern build tooling (Vite)
